## Reporting a Vulnerability

We take the security of Ajapopaja Build seriously. If you discover a security vulnerability, please do NOT open a public issue. Instead, report it through one of the following channels:

1.  **Direct Email**: (Placeholder - please contact the project maintainer directly)
2.  **GitHub Private Reporting**: If available, use the "Report a security vulnerability" feature on the repository.

Please include as much detail as possible, including:
- A description of the vulnerability.
- Steps to reproduce.
- Potential impact.
- Any suggested remediations.

We will acknowledge your report and work with you to resolve the issue as quickly as possible.
